\chapter{Contador de Palavras}
\hypertarget{index}{}\label{index}\index{Contador de Palavras@{Contador de Palavras}}
\label{index_md_README}%
\Hypertarget{index_md_README}%
Este projeto implementa um {\bfseries{contador de palavras}} que processa um texto, normaliza as palavras (transformando-\/as em letras minúsculas e removendo acentos) e as organiza em ordem alfabética. Ele também conta a frequência de cada palavra no texto de entrada, utilizando a abordagem de {\bfseries{Desenvolvimento Orientado a Testes (TDD)}}.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Tecnologias Utilizadas}{Tecnologias Utilizadas}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Linguagem\+:}} C++
\item {\bfseries{Framework de Testes\+:}} Catch2
\item {\bfseries{Ferramentas de Análise\+:}}
\begin{DoxyItemize}
\item cpplint\+: Para verificar o estilo de código.
\item cppcheck\+: Para análise estática do código.
\item Valgrind\+: Para detecção de vazamentos de memória.
\item gcov\+: Para análise de cobertura de testes.
\end{DoxyItemize}
\item {\bfseries{Documentação\+:}} Doxygen (para gerar documentação automática do código)
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_autotoc_md2}
O programa executa as seguintes tarefas\+:
\begin{DoxyItemize}
\item {\bfseries{Transformar em Minúsculas\+:}} Converte todas as letras do texto para minúsculas.
\item {\bfseries{Remover Acentos\+:}} Substitui caracteres acentuados por suas versões não acentuadas (ex.\+: "{}á"{} para "{}a"{}).
\item {\bfseries{Contar Palavras\+:}} Separa e conta a frequência de cada palavra no texto.
\item {\bfseries{Ordenar Palavras\+:}} Ordena as palavras em ordem alfabética, ignorando acentos.
\item {\bfseries{Ler Arquivo\+:}} Permite ler o conteúdo de um arquivo de texto e processá-\/lo.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Como Executar}{Como Executar}}\label{index_autotoc_md3}
